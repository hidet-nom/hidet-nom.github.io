WorkSpacesへの接続

この章では、WorkSpacesに接続するために必要なWorkSpaces クライアントのインストール、接続設定を行います。

1-2-1. WorkSpaces クライアントのインストール

  1. WebブラウザでWorkSpaces クライアントサイトにアクセスします。Windows用クライアントをダウンロードします。

  2. ダウンロードしたAmazon+WorkSpaces.msiを実行し、下の画面で「Next」 をクリックします。

  3. そのまま、下の画面で「Next」 をクリックします。

  4. そのまま、下の画面で「Next」 をクリックします。

  5. 下の画面で「Install」 をクリックします。

  6. 「ユーザーアカウント制御画面」が表示された場合は、「はい」 をクリックします。（ユーザー名、パスワードが必要な場合があります。）

  7. インストールが完了しました。「Finish」 をクリックします。

1-2-2. WorkSpaces クライアントの設定

  1. クライアント端末のWindowsメニューを開き「Amazon WorkSpaces」を選択し、クリックします。

  2. 【この手順は省略可能です。】もし、このような画面が表示された場合は、「Skip this version」をクリックします。

  3. 【すでに日本語表記の場合はこの手順を省略します。】もし、表記が英語で表記を日本語に変更したい場合は、WorkSpacesクライアントの右上の歯車アイコンをクリックし、「Advanced Settings」をクリックします。

  4. 【すでに日本語表記の場合はこの手順を省略します。】日本語を選択し、「Save」をクリックします。

  5. 登録コードを入力し、

    項目 入力内容 備考
    登録コード wsnrt+AJPD3S -

    右下の「登録」をクリックします。

    登録コードは送信メールやAWS マネジメントコンソールで確認できます。

  6. 以下の通り入力し、右下の「サインイン」をクリックします。

    項目 入力内容 備考
    ユーザー名 labuserXX-01 (XXは割当番号) XXは01 ~ 15
    パスワード Amazon!23 -

  7. このような画面が表示されたら右下の「はい」をクリックします。

  8. WorkSpaces(仮想デスクトップ)画面が表示されるまでしばらく待ちます。次の章に進みます。

    実行モードがAutoStopでWorkSpacesの状態がStopped（電源オフ）の場合、WorkSpaces画面が表示されるまで、約2 分かかります。 AlwaysONの場合や切断状態への再接続は、30秒程度で画面が表示されます。

1-2-3. WorkSpaces への接続

  1. WorkSpaces(仮想デスクトップ)画面が表示されました。次の手順に進みます。

  2. WorkSpaces(仮想デスクトップ)画面内で、下のようなポップアップ画面が表示されたら右の「いいえ」をクリックします。

  3. WorkSpaces(仮想デスクトップ)画面を全画面表示にしたい場合、WorkSpaces(仮想デスクトップ)画面ウィンドウで、下のように「表示」「全画面表示」をクリックします。

  4. WorkSpaces(仮想デスクトップ)画面を全画面表示を中止したい場合、マウスポインターを画面最上部に合わせしばらく待つと下画面のようにメニューバーが表示されるので、下のように「表示」「全画面を終了」をクリックします。

  5. WorkSpaces(仮想デスクトップ)画面をウィンドウ化(全画面表示を中止)して、WorkSpaces(仮想デスクトップ)画面ウィンドウの右上の「✖」をクリックします。

  6. 次の演習で使用するため下の画面を表示したままにします。(再接続ボタンはまだ押さないでください。)
