MetaDefender Core の初期設定

1-2-1. MetaDefender Core の初期設定

この章では、MetaDefender Core の初期設定を行います。

  1. デスクトップ上のMetaDefender Coreショートカットを右クリックし、「プログラムから開く」を選択します。

  2. 「Google Chrome」選択し「常にこのアプリを使う」にチェックして「OK」 をクリックします。

  3. このような画面が表示されたら「ログインしない」 をクリックします。

  4. 「Start」 をクリックします。

  5. 「I accept the terms in the License Agreement」にチェックし「Continue」 をクリックします。

  6. 下の表示従いに入力後「Create」をクリックします。

    項目 入力内容 備考
    User name opswat
    Display name OPSWAT
    Password Opswat!23
    Confirm password Opswat!23
    Email training@opswat.local

  7. ライセンスアクティベートを実施します。「Online activation」選択後、「Activation key」を下の様に入力後「Activate」をクリックします。

    項目 入力内容 備考
    Activation key sAyk-gNXp-YmnY-Esxt-bMYi-9bdF-7PXw-eEhd

  8. 「Skip」 をクリックします。

  9. 「Yes」 をクリックします。

  10. 「Finish」 をクリックします。

これで「1-2-1. MetaDefender Core の初期設定」の初期設定は完了です。
