MetaDefender Managed File Transfer (MFT) へのファイルアップロード

この章では、MetaDefender Managed File Transfer (MFT) へのファイルアップロードを行います。

3-4-1. MetaDefender Managed File Transfer (MFT) へのファイルアップロード

  1. デスクトップショートカットの「MetaDefender Managed File Transfer」をダブルククリックします。

  2. MetaDefender Managed File Transfer管理コンソールに以下の表に従って入力し、「Sign In」をクリックします。

    項目 入力内容 備考
    Username opswat
    Password Opswat!23456

  3. 左ペインの「概要」「個人ファイル」を選択し「ファイルをアップロード」をクリックします。

  4. 「ファイルを追加」をクリックします。

  5. アドレスバーに「C:\OPSWAT_Training\demo files」と入力し「demo.pdf」選択後「開く」をクリックします。

  6. 「アップロード」をクリックします。

  7. 状態が「スキャン中」から「無害化済み」になるまでしばらく待ちます。

  8. ファイル名をクリックし詳細なレポートを確認します。

  9. MetaDefender Core管理コンソールにログインし、左ペインの「History」「Processing」を選択しmftユーザーから送られてきた処理を確認します。

「3-4-1. MetaDefender Managed File Transfer (MFT) へのファイルアップロード」はこれで完了です。
