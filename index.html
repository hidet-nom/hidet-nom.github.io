0-1. 本ドキュメントの位置づけ

WorkSpaces利用を検討している方向けハンズオンの解説資料です。本書の記載範囲については、本ガイドの範囲 に記載しています。

0-2. 本ガイドの範囲

本ワークショップでは、以下の3つについてハンズオンを実施します。

  1. AWS Basics: クラウドインフラの基礎となるVPC、サブネット、EC2インスタンスなどの構築を通して、AWSの基礎を体験するハンズオンです。
  2. AWS CDK CDKを利用した構築自動化（Infrastructure as Code）のハンズオンです。
  3. Amazon WorkSpaces:　AWSが提供するVDIサービス「Amazon WorkSpaces」について、構築、管理などの操作を体験いただくハンズオンです。
  4. 分析Sandobox環境構築: デジタルサービス推進部で設計したSandobox環境の環境操作を体験いただくハンズオンです。

0-3. ハンズオンの前提

今回のワークショップでは、ハンズオンに必要な環境（端末およびAWSアカウント）をAWSにて準備致します。 端末を持ち込んで利用する方は、ブラウザとWorkSpaces クライアントをご用意ください。

0-4. 本書の対象者

本書は、AWS環境の利用を検討いただける方を想定して作成しています。AWSを知らない方、AWSを触ったことがない方向けの入門レベルです。
